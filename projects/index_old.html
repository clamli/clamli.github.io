#1 CALT

CALT, collaborative active learning tree, is a recommender system proposed for cold-start item recommendation.

In CALT, multiple decision trees are built with nodes associated with user clusters. Starting from the root node, a new item is adaptively presented to different clusters of users according to query result from the previous user cluster. When no rating is elicited during the query, content-based collaborative filtering is employed to estimate the ratings.

CALT is distinguished by making use of both content information and queried rating to carry out more accurate recommendation for cold-start items.



#2 MiniSQL

A small query tool implementing the basic functions of MySQL.

MiniSQL has a subset of commands of MySQL, including INSERT, SELECT, DELETE, UPDATE, CREATE/DROP TABLE, CREATE/DROP INDEX.

MiniSQL is mainly divided into six modules: interpreter, API, catalog manager, record manager, buffer manager and index manager. Interpreter is achieved by Antlr4, a powerful parser generator, and is used to analyze the input commands. Catalog manager helps to manage header information of tables. Record manager deals with all the records in a table and is tightly related with buffer manager, which is a small cache (blocks set) used to temporarily hold information and write them back to the disk or read them out from the disk whenever required. With the buffer, we can avoid constant operations of reading/writing data out from/into the disk and thus save time. Index manager is implemented by B+ Tree. It use a tree structure to sort chosen columns of tables so that future request for these data can be done in O(logN) time complexity, instead of O(N).



# 3 RoomBreak

RoomBreak is a game implemented by C++ and the scene of the game is built by Maya.

This work mainly contains the modules for Scene import, View Control, Collision Detection, Click Event Control, Music Control, Light Control and Texture Control. Scene import module is used to import maya-built house and all the background items. View Control module is used to change the position when people move. Collision Detection module is based on the Cartesian coordinate system and it saves a set of coordinates to for each item in order to represent their border. Click Event Control module helps to change the scene whenever a click happens and music control determines the appropriate song for each stage. Light Control and Texture Control are based on the functions provided by Open Graphic Library (OpenGL) and they are used to render the scene and make it real.
